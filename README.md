## 📌 Project Overview
This project is an **end-to-end Excel-based Sales Analysis Dashboard** inspired by Swiggy’s food delivery business.  
The objective is to transform raw data into **actionable business insights** using Excel dashboards, KPIs, and slicers.

The dashboard provides a quick view of **revenue performance, customer behavior, ratings, food preferences, city-wise demand, and state-wise sales trends**.

---    

## 🎯 Key Objectives
- Analyze total revenue and order performance    
- Track monthly, weekly, and daily sales trends    
- Understand customer ratings and average order value      
- Compare Veg vs Non-Veg sales contribution      
- Identify top-performing cities and states       
- Enable interactive analysis using slicers

## 📈 Dashboard Insights

### 🔹 Sales Trends
- **Monthly Sales** peak in **May and August**
- **Weekly Trend** shows stable revenue with weekend uplift
- **Daily Sales** are highest on **Saturday and Thursday**

### 🔹 Food Type Contribution
- **Non-Veg**: ₹ 34.2M (64%)
- **Veg**: ₹ 18.8M (36%)

➡️ Non-Veg food items contribute the majority of revenue.

### 🔹 Top 5 Cities by Sales
1. **Bengaluru** – ₹ 5.46M  
2. **Lucknow** – ₹ 3.12M  
3. **Hyderabad** – ₹ 3.02M  
4. **Mumbai** – ₹ 3.02M  
5. **New Delhi** – ₹ 2.83M  

### 🔹 State-wise Sales Analysis
- Strong sales concentration in **South and Central India**
- State-wise revenue ranges from **₹ 0.56M to ₹ 5.46M**
